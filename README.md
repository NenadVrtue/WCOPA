## ðŸ§ž Commands

All commands are run from the root of the project, from a terminal:

| Command                   | Action                                           |
| :------------------------ | :----------------------------------------------- |
| `npm install`             | Installs dependencies                            |
| `npm run dev`             | Starts local dev server at `localhost:4321`      |
| `npm run build`           | Build your production site to `./dist/`          |
| `npm run preview`         | Preview your build locally, before deploying     |
| `npm run astro ...`       | Run CLI commands like `astro add`, `astro check` |
| `npm run astro -- --help` | Get help using the Astro CLI                     |

## Steps to get started after cloning the template

- Add new projectId and project dataset to astro.config.mjs and env files
- Add site url, project name, project title and optionally new icon(logo) in sanity.config.ts
- Define singleton schema, temporarily disable singleton filtering so you can add documents for singletons
- Make sure you copy documents id, and add it correctly to the structureTool in sanity.config
